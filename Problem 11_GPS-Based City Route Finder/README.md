# 🚗 GPS-Based City Route Finder using A* Algorithm

A professional Python desktop application built with **Tkinter** that finds the **optimal route between cities/nodes** using the powerful **A* (A-Star) Search Algorithm**.  

This project simulates a GPS navigation system where users can enter roads, travel costs, heuristic values, and calculate the shortest intelligent route.

---

## 📌 Project Overview

The **A* Search Algorithm** is widely used in:

- 🗺️ GPS Navigation Systems  
- 🚗 Route Optimization  
- 🤖 Robotics Path Planning  
- 🎮 Game AI Movement  
- 📦 Logistics & Delivery Routing  

This application provides an interactive GUI where users can test A* search visually.

---

## ✨ Features

✅ User-friendly GUI using Tkinter  
✅ Enter custom graph edges and travel costs  
✅ Add heuristic values for smart searching  
✅ Select Start Node and Goal Node  
✅ Finds **Optimal Path** instantly  
✅ Displays Total Route Cost  
✅ Shows Explored Nodes during search  
✅ Clean modern dark theme interface  

---

## 🧠 Algorithm Used

### A* Search Algorithm

A* combines:

- **g(n)** = Actual cost from start node  
- **h(n)** = Estimated cost to goal node  
- **f(n) = g(n) + h(n)**

It always chooses the most promising path first.
