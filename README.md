# 🧩 Sudoku Solver using CSP Algorithm

### 🚀 Intelligent Sudoku Puzzle Solver using Artificial Intelligence

**Constraint Satisfaction Problem (CSP)** with **Backtracking Search Algorithm**.

## 📌 About The Project

Sudoku is one of the most popular logic puzzles in the world.  
This project uses **Artificial Intelligence techniques** to solve Sudoku automatically.

The system allows users to:

✅ Play Sudoku manually  
✅ Check entered answers  
✅ Generate random puzzle boards  
✅ Auto-solve puzzle using CSP

# 🚗 Smart Navigation System using BFS, DFS & A* Algorithm

### 🚀 Intelligent Path Finding System using Artificial Intelligence

A professional Python desktop application that finds the best route between cities/nodes using  
**Breadth First Search (BFS)**, **Depth First Search (DFS)** and **A* Search Algorithm**.

<br>

![Python](https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python)

</div>

## 📌 About The Project

Navigation systems are used to find the shortest and smartest route between locations.

This project uses **Artificial Intelligence search algorithms** to solve path finding problems efficiently.

The system allows users to:

✅ Enter graph connections manually  
✅ Provide heuristic values  
✅ Choose Start and Goal nodes  
✅ Find shortest route instantly  
✅ Learn AI search algorithms practically
