# 🧩 Sudoku Solver using CSP Algorithm

### 🚀 Intelligent Sudoku Puzzle Solver using Artificial Intelligence

A professional Python desktop application that solves Sudoku puzzles using  
**Constraint Satisfaction Problem (CSP)** with **Backtracking Search Algorithm**.

## 📌 About The Project

Sudoku is one of the most popular logic puzzles in the world.  
This project uses **Artificial Intelligence techniques** to solve Sudoku automatically.

The system allows users to:

✅ Play Sudoku manually  
✅ Check entered answers  
✅ Generate random puzzle boards  
✅ Auto-solve puzzle using CSP  
✅ Learn AI search techniques practically
